De forma a correr os programas devem ser executados os seguintes comandos na linha de comandos/shell:
python Q1.py
python Q2.py

Ao executar o Q1.py são gerados um ficheiro q1.fasta que contém a população inicial e um q1_output.txt com os resultados da hamming distance e Jukes-Cantor para cada geração. Este último ficheiro é utilizado para gerar o plot em Matlab com o script makePlot.m.

Ao executar o Q2.py são gerados um ficheiro q2.fasta que contém a população inicial e diversos ficheiros fasta q2_1.fasta, ..., q2_5.fasta
 que contém a população final correspondente a cada uma das perguntas.
